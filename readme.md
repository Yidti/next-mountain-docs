**Next Mountain** 是一個登山戶外文件網站，希望大家都能夠平安下山。

![](/public/demo.png)


## 安裝pnpm
```
curl -fsSL https://get.pnpm.io/install.sh | sh -
```
## 架設網站
```
pnpm install
pnpn dev
```

## Development
